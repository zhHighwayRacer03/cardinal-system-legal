# Datenschutzerklärung

Letzte Aktualisierung: 26. März 2025

Diese Datenschutzerklärung informiert darüber, wie der Discord-Bot **Cardinal System** („der Bot“) personenbezogene und serverbezogene Daten verarbeitet und speichert.

1. Welche Daten werden gespeichert?

Der Bot speichert ausschließlich technische und serverbezogene Informationen, darunter:

#

Server-Konfigurationen (`/configs/[SERVER_NAME]_config.json`)
  • Server-ID, Servername, Mitgliederzahl
    • Owner-ID und -Benutzername
      • Channel-IDs (z. B. Log, Ticket, Warnung)
        • Rollen-IDs (z. B. Admin, Mod, Ticket)
          • Online-Benachrichtigungseinstellungen
            • Einladungslink und Erstellungsdatum

              • #

              Verwarnungen (`/warnings/[SERVER_NAME]_warnings.json`)
              • Servergrundinformationen (ID, Name, Owner, Mitgliederzahl)
                • Verwarnungen pro Nutzer mit:
                  • - Grund der Verwarnung
                  - Moderator-ID
                  - Zeitstempel

                  #

                  Befehlsinformationen (`Command List.json`)
                  • Auflistung aller Bot-Befehle, gruppiert nach Kategorien
                    • Erforderliche Berechtigungen
                      • Beschreibung der Funktionen

                        • #

                        Temporäre Musikdaten
                        • Aktueller Song
                          • Stage-Kanal-Status
                            • Song-Info-Konfiguration

                              • #

                              **Nicht gespeichert werden:**
                              • Nachrichteninhalte
                                • Bilder oder Medien
                                  • Private Nachrichten
                                    • Passwörter oder Tokens
                                      • Persönliche Informationen über IDs hinaus

                                        • Alle Daten werden lokal im Dateisystem in JSON-Dateien gespeichert – getrennt pro Server.

                                        2. Wie lange werden die Daten gespeichert?

                                        • Die Daten werden dauerhaft gespeichert, solange der Server aktiv mit dem Bot arbeitet.
                                          • Sie bleiben erhalten, bis sie über Bot-Befehle (`/clear`, `/clearall`) oder manuell gelöscht werden.
                                            • Beim Entfernen des Bots vom Server können Daten auf Anfrage gelöscht werden.

                                              • 3. Wer hat Zugriff auf die Daten?

                                              Zugriffsberechtigung besteht ausschließlich für:

                                              • **Bot-Administratoren:** Vollzugriff auf Konfigurationen und Warnungen
                                                • **Bot-Moderatoren:** Eingeschränkter Zugriff auf Verwarnungsdaten
                                                  • **Bot-Entwickler:** Zugriff auf Rohdaten im Dateisystem (JSON-Dateien)

                                                    • **Normale Mitglieder** eines Discord-Servers haben keinen Zugriff auf diese Daten. Alle Funktionen sind durch das Discord-Berechtigungssystem geschützt.

                                                    4. Deine Rechte

                                                    • Du hast das Recht, Auskunft über die über dich gespeicherten Daten zu erhalten.
                                                      • Du kannst eine Löschung deiner Daten beantragen.
                                                        • Anfragen zur Einsicht oder Löschung bitte über den Support-Discord:
                                                          • [https://discord.gg/XUDkgETzS4](https://discord.gg/XUDkgETzS4)

                                                          5. Änderungen

                                                          Diese Datenschutzerklärung kann jederzeit aktualisiert werden. Bei wesentlichen Änderungen wird dies im Bot oder über den Support-Server kommuniziert.
